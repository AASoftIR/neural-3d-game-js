# Contributing

This repository uses a restrictive source-available license.

Contributions are welcome only if you agree that your contribution may be used by the project author under the repository license terms.

## Rules

- Do not submit copied assets.
- Do not submit code taken from incompatible licenses.
- Keep the project dependency-free unless there is a strong reason.
- Keep `index.html` directly runnable.
- Do not remove author/license notices.
- Do not add telemetry, tracking, ads, or external network calls.

## Good Contributions

- performance improvements
- bug fixes
- better mobile controls
- accessibility settings
- level design improvements
- cleaner code organization
- README/documentation improvements
- visual polish that keeps the project lightweight

## Development

Run locally:

```bash
python3 -m http.server 8080
```

Open:

```txt
http://localhost:8080
```

## Debug

Use:

```txt
F3
```

to show performance and entity counters.
